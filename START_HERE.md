## ⚡ Start Using It Right Now

### Step 1: Install (60 seconds)
```bash
cd /workspaces/Gig-Setlist-Manager
npm run install-all
```

### Step 2: Run (30 seconds)
```bash
npm run dev
```

Open your browser to: **http://localhost:3000**

### Step 3: Try It Out (2 minutes)
1. Click "New Setlist" 
2. Add some songs
3. Click "📥 Export CSV" → Your file downloads!

## ❓ Quick FAQs

**Q: Do I need to set up Google Cloud to use the app?**  
A: No! CSV export works immediately. Google Sheets is completely optional.

**Q: Where is my data stored?**  
A: In a SQLite database locally in the `backend/data/` folder. It's yours forever.

**Q: Can I export my data?**  
A: Yes! Click the 📥 button and get a CSV file that works in Excel, Google Sheets, or anywhere else.

**Q: How do I back up my setlists?**  
A: Export to CSV - that's your backup! Download whenever you want.

**Q: Can I share setlists with my band?**  
A: Export as CSV and email it! They can import it into their own manager.

**Q: Is it free?**  
A: Yes! It's a local web app you can run on your computer.

**Q: Can I use it on my phone?**  
A: Yes! The design is responsive. Just run the dev server and access from your phone.
